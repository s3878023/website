WHAT IS GIT AND GITHUB?

Git is a famous control system that was developed back in 2005, git is installed and maintained on the user’s local system and enable the coder to record all of your present program.

Git can be in used even there is no connection to the Internet. (devmountain n.d).

Git when being compared to other control systems, Git shows that it is very responsive and easy to use, especially Git is free to use. This system is designed to work smoothly with txt files which is the true nature of code. (devmountain n.d)

Github, on the other hand, is designed to be a Git repository hosting device. Through GitHub, you can share your code with others and encourage them to develop and upgrade on your previous Git branches. As changes happened, new branches will be created, enabling all of the teammates to continue to code and avoid overwriting each other’s work. (devmountain n.d)
