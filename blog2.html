BASIC GIT COMMAND.

Here are a few easy steps for you to install Git: So there are two main OS is Mac and Window:

Mac: The easiest way is to install the Xcode Command Line Tools and simply do it by running git from the Terminal the very first time. If you don’t have it installed already, it will prompt you to install it. If you want a more up to date version, you can also install it via a binary installer. A macOS Git installer is maintained and available for download at the Git website, at link

Github, on the other hand, is designed to be a Git repository hosting device.
